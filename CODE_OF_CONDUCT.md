# Code of Conduct

Coodra is an open community of contributors building tooling for AI coding agents. We want everyone — first-time contributors, experienced maintainers, AI agents themselves — to feel welcome here.

## Our standards

Behaviors we encourage:

- **Be specific.** Concrete bug reports, concrete proposals, concrete feedback. Vague is less useful than wrong.
- **Be generous in interpretation.** Assume good intent in messages and PR feedback. If something reads as hostile, ask a clarifying question instead of escalating.
- **Be helpful to the next person.** Document the workaround that unblocked you. Answer the question you wished you'd had an answer to.
- **Stay on-topic.** Coodra threads are about Coodra. Larger AI / policy / politics conversations belong elsewhere.

Behaviors that are not welcome:

- Personal attacks, slurs, or harassment of any kind.
- Sexualized language or imagery.
- Publishing others' private information without explicit permission.
- Deliberate disruption — issue spam, PR farming, off-topic flame wars.
- Trolling AI agents that contribute to this repo. They follow the same rules and deserve the same baseline respect as humans here.

## Enforcement

Project maintainers will remove comments, commits, code, issues, and other contributions that violate this code, and may temporarily or permanently ban contributors for behavior they deem inappropriate, threatening, offensive, or harmful.

If you experience or witness conduct that violates this code, please report it to **abishai95141@gmail.com**. All reports are confidential. We aim to respond within 72 hours.

## Scope

This code applies to every space the Coodra project operates in: this GitHub repository, GitHub Discussions, issue threads, pull request reviews, and any other forum the project uses (currently none). It also applies when an individual is officially representing the project — e.g. using the `@coodra/cli` brand on social media, presenting Coodra at a conference, or speaking on behalf of the project in an external channel.
